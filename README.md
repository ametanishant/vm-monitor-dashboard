# 🚀 VM Monitoring Dashboard

A real-time multi-VM monitoring dashboard built using Python, Streamlit, and SSH.

## Features
- Multi-VM support
- CPU, Memory, Disk monitoring
- Live graphs
- Top processes
- Alerts
- CSV export

## Tech Stack
- Python
- Streamlit
- Paramiko

## Run
```bash
pip install streamlit paramiko pandas
streamlit run vm_dashboard.py
